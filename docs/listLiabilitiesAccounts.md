# Lists all liability accounts

**URL** : `/accounts/liability`

**Example**: `http://127.0.0.1:8080/accounting-system-ddd/api/accounts/liability`

**Method** : `GET`

## Success Response

**Code** : `200 OK`

 
**Example**

````json

[
    {
        "id": 1,
        "accountNumber": 9887777665,
        "accountName": "Test account",
        "currentBalance": -1000
    }
]

````
