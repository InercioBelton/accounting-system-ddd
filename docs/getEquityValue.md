# Gets the equity value

**URL** : `/accountingJournal/equityValue`

**Example**: `http://127.0.0.1:8080/accounting-system-ddd/api/accountingJournal/equityValue`

**Method** : `GET`

## Success Response

**Code** : `200 OK`
 
**Example**

````json
{
    "code": 200,
    "message": "SUCCESS",
    "description": "Equity value is 0.00"
}

````
