# Gets the assets value

**URL** : `/accountingJournal/assetsValue`

**Example**: `http://127.0.0.1:8080/accounting-system-ddd/api/accountingJournal/assetsValue`

**Method** : `GET`

## Success Response

**Code** : `200 OK`
 
**Example**

````json
{
    "code": 200,
    "message": "SUCCESS",
    "description": "Assets value is 1000.00"
}

````
