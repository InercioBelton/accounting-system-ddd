# Gets the liabilities value

**URL** : `/accountingJournal/liabilitiesValue`

**Example**: `http://127.0.0.1:8080/accounting-system-ddd/api/accountingJournal/liabilitiesValue`

**Method** : `GET`

## Success Response

**Code** : `200 OK`
 
**Example**

````json
{
    "code": 200,
    "message": "SUCCESS",
    "description": "Liabilities value is -1000.00"
}

````
